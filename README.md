# Yan Browser — ✦ YOUR BROWSER · YOUR RULES

**A private, offline-first mobile browser built for the real world.**

> *"Like Firefox built Gecko — a browser built for the people, not for corporations."*

---

## 🌏 Who is this for?

This browser was built for **Indonesia** and **Asia** — where internet is slow, expensive, or simply unavailable.

- 270M+ people in Indonesia, mostly mobile-first
- Millions with limited or no reliable internet access
- Anyone who values **privacy** and **independence** from Big Tech

**If you have ever lost access to a site because of bad signal — this is for you.**

---

## 📱 What is Yan Browser?

Yan Browser lets you **download websites once** and access them **forever** — without internet.

- Download sites from a curated catalog
- Open them instantly, offline, anytime
- No Google. No tracking. No ads. No limits.

Think of it like a personal library of the web — yours to keep.

---

## ✦ Features

- **Offline-first** — downloaded sites work without internet
- **Multi-server** — connect to any approved manifest server
- **Full-text search** — supports EN / ID / ZH / KO
- **Live clock** — with timezone selection (default: Jakarta, WIB)
- **OTA updates** — silent app updates in the background
- **Page cache** — faster loading on revisit
- **Gemini AI** — optional AI search assistant
- **Dark mode** — professional, minimal UI

## 📦 How to Build

```bash
git clone https://github.com/YanMing488/YanBrowser.git
cd YanBrowser
npm install
npx expo prebuild --platform android
cd android && ./gradlew assembleRelease
```

Or download the latest APK from [Releases](../../releases).
